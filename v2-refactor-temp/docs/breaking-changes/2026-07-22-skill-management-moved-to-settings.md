## What changed

Global Skill management moved from the Agent session display menu to Settings → Skills, directly below MCP.

## Why this matters to the user

The Agent session menu no longer includes Manage Skills. Online search, local and system imports, Skill details, and uninstall actions are now available from the new Skills settings page.

## What the user should do

Use Settings → Skills to manage installed Skills. Agent-specific Skill enablement remains available from the Agent configuration dialog.

## Notes for release manager

No Skill data or enablement state is migrated; only the global management entry point changed.
