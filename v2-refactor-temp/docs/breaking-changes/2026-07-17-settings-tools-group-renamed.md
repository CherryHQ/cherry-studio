## What changed

The Capabilities & Tools group in Settings is now named Tools. Environment Dependencies remains in the System group, directly below System.

## Why this matters to the user

Users will see a shorter Tools group label while runtime dependencies remain with system settings.

## What the user should do

Nothing — the settings and their saved configuration are unchanged.
