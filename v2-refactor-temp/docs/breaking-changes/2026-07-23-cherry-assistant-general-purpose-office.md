## What changed

Cherry Assistant is now a general-purpose assistant instead of only a Cherry Studio product helper. It can also create content-rebuilt DOCX, PDF, PPTX, and XLSX files from supported text and tabular sources.

## Why this matters to the user

Users can ask Cherry Assistant to complete ordinary writing, analysis, translation, coding, document, presentation, and data tasks directly in an Agent session. Office conversions rebuild content and do not preserve advanced source formatting, macros, animations, or formulas.

## What the user should do

Nothing - the new capabilities are available automatically. Use a specialized tool when high-fidelity Office editing or scanned-PDF OCR is required.
