## What changed

Cherry Assistant is now included in new and existing Agent libraries unless the user previously deleted it. It starts in Auto-edit Mode, which allows file edits without confirmation while still asking before commands run. It is also a general-purpose assistant instead of only a Cherry Studio product helper, and can create content-rebuilt DOCX, PDF, PPTX, and XLSX files from supported text and tabular sources.

## Why this matters to the user

Existing users will see Cherry Assistant appear alongside their current Agents. They can ask it to complete ordinary writing, analysis, translation, coding, document, presentation, and data tasks directly in an Agent session. Office conversions rebuild content and do not preserve advanced source formatting, macros, animations, or formulas.

## What the user should do

Nothing - the new capabilities are available automatically. Use a specialized tool when high-fidelity Office editing or scanned-PDF OCR is required.
