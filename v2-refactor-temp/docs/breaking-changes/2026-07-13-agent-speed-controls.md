## What changed

Agent sessions now configure reasoning effort and Fast mode through a model-aware speed control beside the composer. The available effort levels and defaults follow the selected model.

## Why this matters to the user

Users will find Agent reasoning controls in a different place, and the choices can change when they switch models. Fast mode appears only for models that support it.

## What the user should do

Nothing - this is automatic. Open the speed control in the Agent composer when a different reasoning effort or Fast mode is needed.
