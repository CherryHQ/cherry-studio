## What changed

OCR configuration no longer appears inside Document Processing. It now has its own OCR entry directly below Document Processing in the Tools section of Settings.

## Why this matters to the user

Users configuring image text recognition should open Settings → Tools → OCR. Document Processing now lists only document-to-Markdown processors.

## What the user should do

Nothing — existing OCR defaults and processor configuration are unchanged.
