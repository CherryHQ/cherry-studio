# Cherry Studio Chat

Cherry Studio Chat covers the product language for the ordinary conversation experience. Agent-specific chat surfaces are separate unless explicitly named.

## Language

**Chat Right Pane**:
The auxiliary panel attached to the ordinary chat area for topic-scoped views such as conversation resources, branch flow, and trace inspection.
_Avoid_: Agent Right Pane, generic right panel

**Pane Entry**:
A top-bar control outside the Chat Right Pane that opens, switches, or closes a specific right-pane view. It may be an icon-only entry or a compact count entry for conversation resources.
_Avoid_: Shortcut icon, keyboard shortcut, tab button

**Right-Pane View**:
One selectable content surface inside the Chat Right Pane.
_Avoid_: Tab, panel, page

**Chat Right Pane Header**:
The control row inside the Chat Right Pane that contains pane-level actions such as maximize and close.
_Avoid_: Tab bar, navbar, shortcut row
