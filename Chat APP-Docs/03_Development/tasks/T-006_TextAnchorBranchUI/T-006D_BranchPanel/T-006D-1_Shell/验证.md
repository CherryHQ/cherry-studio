### 期望流程

1. 新建 topic + 发送一条消息（用任意非 Ollama provider）
2. 在 AI 回复任意一段同段落文字上**选中**，右键
3. 看到 4 个菜单项：Copy / Quote / **Ask about this** / **Open as branch**
4. **点击 "Open as branch"** 或 **"Ask about this"** → Dialog 弹出，显示：
   - 头部：`Branch from selection` 标题
   - 选区引用块（黄边竖线 + italic）
   - mono 小字：`messageId: <UUID>` / `blockId: <UUID>`
   - 追问 textarea（label 「Follow-up question」）
   - 底部按钮：`Cancel` / `Create branch`
5. **取消按钮**：点击 → Dialog 关闭
6. **创建分支按钮**：
   - 不输入直接点击 → Dialog 关闭 + DevTools Console 显示：
     ```
     [BranchPanel] branch-panel: create branch (placeholder, T-006D-2 wires fork) { messageId, blockId, selectedText, followUp: '' }
     ```
   - 输入文字后点击 → 同上但 `followUp` 含输入文字
7. **再次右键打开 panel**：上次输入的 followUp 应该已清空（关闭时 effect 清）
