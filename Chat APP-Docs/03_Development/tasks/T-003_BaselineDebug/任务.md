## 结论
**根因**：renderer `AssistantService.ts:77-89` 合成 `id: 'default'` sentinel；v2 后端按 UUID 校验 FK；fresh install 路径不经过 `AssistantMigrator` 的 sentinel 重映射。

**推荐**：方案 B（`mapLegacyTopicToDto` 把 'default' 转 null），一行改动，符合 v2 nullable schema。
