### DevTools Console 端到端断言（一键脚本）

```js
;(() => {
  const state = window.store?.getState?.()
  const msgs = Object.values(state?.messages?.entities ?? {})
  const blocks = Object.values(state?.messageBlocks?.entities ?? {})
  const assistantMsgs = msgs.filter(m => m?.role === 'assistant')
  console.table(assistantMsgs.map(m => ({
    id: m.id?.slice(0, 8),
    status: m.status,
    blocksCount: m.blocks?.length ?? 0,
    blockIds: (m.blocks ?? []).map(id => id.slice(0, 8)).join(',')
  })))
  console.table(blocks.map(b => ({
    id: b?.id?.slice(0, 8),
    type: b?.type,
    status: b?.status,
    contentPreview: b?.content?.slice?.(0, 40)
  })))
})()
```

期望（修完之后）：
- 所有 assistant message 的 status 都是 `'success'`（不是 `'pending'`/`'processing'`）
- 每条 assistant message 的 blocksCount ≥ 1
- 对应 block id 在 messageBlocks 表里能找到，type 是 `'main_text'`（或多块时含 `'main_text'`），status 是 `'success'`
