### 风险与未决

- 部分行号来自 Explore agent 抽样，未逐一复核；具体修改前请 grep 一次。
- `migrate.ts` 显示 206 版迁移函数，单纯升级 redux-persist 状态结构；与 v2 数据迁移器（`ChatMigrator`）是两套机制，不要混淆。
- 聊天 UI 的 antd 用量很大（agent 报告 53 处 antd import），完整切换到 `@cherrystudio/ui` 工作量不小。
