# FAQ Collector

**收录标准**: 通用性高、有明确方案、可复现、非个人偶发情况。不收录: 一次性细节、已有相同条目、未解决问题。

**文件**: `<project_root>/.skill-creator/faq.md`（目录不存在则 `mkdir -p .skill-creator` 创建）

**条目格式**（追加到末尾）:
```markdown
### Q: [通用化问题表述]
**A:** [简洁方案]
[分步骤操作]
- **关键词**: [逗号分隔]
- **场景**: [新手引导 / 描述触发 / 输出格式 / skill 调试 等]
- **收录日期**: YYYY-MM-DD
---
```

**流程**: 问题解决→判断收录标准→读 FAQ 查重→无重复则通用化后追加→有相似但更好则更新

**搜索匹配**: 用户提问时先读 FAQ 关键词匹配→命中直接给答案→未命中走正常引导流程
